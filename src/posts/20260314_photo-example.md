블로그 포스트에 사진을 넣는 방법은 간단합니다. 마크다운 이미지 문법을 사용하면 됩니다.

## 외부 이미지 URL 사용

외부 URL을 직접 사용할 수 있습니다.

![샘플 이미지](https://picsum.photos/800/400)

## 문법

마크다운에서 이미지를 넣는 문법은 다음과 같습니다.

```markdown
![대체 텍스트](이미지 URL)
```

## 여러 장의 사진

여러 장의 사진도 자유롭게 넣을 수 있습니다.

![풍경 1](https://picsum.photos/800/300?random=1)

![풍경 2](https://picsum.photos/800/300?random=2)

이렇게 마크다운 문법만으로 간편하게 이미지를 추가할 수 있습니다.
