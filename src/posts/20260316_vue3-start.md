Vue 3는 프론트엔드 프레임워크로, 컴포넌트 기반의 UI 개발을 지원합니다.

## Composition API

Vue 3에서 가장 큰 변화는 Composition API의 도입입니다. `setup()` 함수 안에서 반응형 데이터와 로직을 구성할 수 있습니다.

## 반응형 데이터

`ref()`와 `reactive()`를 사용하여 반응형 데이터를 선언할 수 있습니다. 데이터가 변경되면 자동으로 화면이 업데이트됩니다.

```js
import { ref } from 'vue'

const count = ref(0)
```
